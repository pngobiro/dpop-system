Statistics Information System

The statistics Information System is capstone project for Moringa School. The project is a web application that will be to collect, store, process and analyze data from the Judiciary. The application will be used by the Judiciary to collect data from the courts. The application will also be used to generate reports and visualizations for the data collected.
